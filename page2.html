Crystal Sage Game

Put your memory skills to the test. Can you defeat the Crystal Sage? Spells will have a random number assigned and it is up to you to figure it out!

This application is a simple game built entirely with Javascript. The main functionality around this game is the use of assigning random numbers to a game. The game will reset if the player goes over the total hit points of the boss giving a loss. Each game will have different numbers assigned to the boss health and spell damage.
