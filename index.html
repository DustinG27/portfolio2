Beer Memory Game

Put your memory skills to the test. Can you click all 15 beers without clicking a duplicate?

This application is a simple game built entirely with React using React State variables. Players will be able to click an image that is tracked through an incremented score. A high score will increment if players reach higher correct clicks. this app also uses a shuffle system to add a challenge to the game.

Find A Drummer

In search for a drummer for the next upcoming show? Or maybe you need a full time member. Come here for all your drummer needs!

This application is a form based app that uses servers with Express and Node.JS. The user enters their name and a profile picture to start. Once the form is filled out the application will match the form will match the closest drummer in the list of drummers. Once submitted the users score, picture, and name will be stored in the drummer list.
